

This is a basic multi-page portfolio website created using only HTML.

## 📌 Project Description

This website contains information about me, my projects, articles, and contact details.  
The purpose of this project is to practice and understand HTML fundamentals and how multi-page websites work.

---

## 📂 Pages Included

*Home (index.html)** – About me and skills
*Projects (projects.html)** – List of projects with table and images
*Articles (articles.html)** – Articles written by me
*Contact (contact.html)** – Contact details and simple contact form

---

## ✨ Features

- Navigation bar on every page
- Multi-page linking
- Project details in table format
- Images included in project section
- Contact form (HTML only)
- Clean and simple layout

---

## 🛠 Technologies Used

- HTML5

---

## 🎯 Purpose of Project

This project was built to:

- Practice HTML structure
- Understand linking between pages
- Learn tables, forms, and images in HTML
